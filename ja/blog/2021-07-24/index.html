GitHub Pagesに移転しました

Jul 24, 2021

元々、個人サイトを Vultrという海外の安価な VPS で運用していました。そういう作業自体は好きなのですが、流石に静的なサイトをホストするためだけに VPS を維持するのは手間がかかりすぎるので、 GitHub Pagesでホストすることにしました。以前に GitHub Pages を使ったときには Jekyll を使ったのですがいまいちしっくり来なかったのと、 Academicというテーマが良さそうだったので、サイトジェネレータは Hugoを採用しました。お金がかかるのを嫌がったくせに、ドメイン名を新たに取得してみたりして。
